# Respositorio sobre la base de datos abiertos direccion General de Epidemiología sobre Influenza , COVID-19 y otros virus respiratorios

Autor: Gustavo de Jesús Escobar Mata

## Introducción 
Este repositorio se trata sobre la base de datos de influenza, covid-19 y otros virus respiratorios. 

## Objetivos
**Objetivo general**
Implementación de algoritmos sobre aprendizaje automatico para la predicción de COVID-19 en México.

## Carpetas
A continuación se muestra la descripción de las carpetas que se encontraran en el repositorio.
- *Galeria*: En esta carpeta se colocaran todas las imagenes generadas.
- *Codigos*: En esta carpeta se colocaran todas las versiónes de los codigos en Python para los metodos de análisis y predictorios sobre la base de datos.
